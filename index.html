Why Butter Cake?

Butter Cake is an open source CSS framework based on Flexbox, Giving Developers Flexibility & less Classes to create components and included 100+ helper Classes to get the task done easily without writing too many lines of css.

How to use


Download Butter Cake and link css file and js file

                
<link rel="stylesheet" href="dist/butterCake.css">

......

<script src="js/butterCake.js"></script>

That's it, now you are ready to go

Form Elements

To use Butter Cake's input style simply add .input class on input element


Simple Input Field

            
<!-- Simple input element -->
<input class="input" type="text" placeholder="John Doe">

Grouping Label and input

            
<!-- Input group -->
<div class="group">
    <label>Name</label>
    <input class="input" type="text" placeholder="John Doe">
</div>

Checkbox and radio group

            
<!-- Checkbox & Radio group -->
<div class="group checkbox-group">
    <input type="checkbox" id="confirmField">
    <label for="confirmField">Check Box</label>
</div>

<div class="group checkbox-group">
    <input type="radio" id="confirmField1">
    <label for="confirmField1">Radio Button</label>
</div>

Modal


modal



To create a basic modal


create a div tag with the class of .modal this will be the main wrapper of the modal and a unique ID for that modal 

            
<div class="modal" id="modal1" ></div>


then create a container inside the modal with the class of .modal-container

            
<!-- MODAL -->
<div class="modal" id="modal1">
    <div class="modal-container modal-sm">

        <!-- header -->
        <div class="modal-header bg-secondary">
            ........
        </div>

        <!-- body-->
        <div class="modal-body">
            ............
        </div>

        <!-- footer-->
        <div class="modal-footer">
            ............
        </div>

    </div>
</div>



Modal Trigger


Use modal-open class to create a trigger & add data-modal="modalID" to point to a modal

When the modal trigger is clicked the modal will open up.

            
<a href="#!" class="btn-primary modal-open" data-modal="modal1">modal</a>


Modal Exit


To close a modal add modal-close class to a button or any element that you want to use as modal closer, then add data-modal="modalID" to point to the modal that you wanna close

When the modal close trigger is clicked the modal will be closed.

            
<div class="modal-close" data-modal="modal1"><i class="fa
fa-close"></i></div>
